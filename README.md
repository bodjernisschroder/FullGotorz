# ?? FullGotorz - Final Exam Project

Welcome to **FullGotorz** — our final exam project that combines a Blazor Web App and a supporting Web API architecture for a complete travel package booking experience!

---

## ?? Technologies Used

- **.NET 8** (C#)
- **Blazor Server**
- **ASP.NET Core Web API**
- **Entity Framework Core**
- **SQLite** (for the TravelBridgeAPI database)
- **REST APIs**
- **HTTPClient** (for API communication)
